# Smallville S.V.6.0 🛡️

Smallville S.V.6.0 is an autonomous, rule-based security orchestration suite designed for rapid reconnaissance and vulnerability assessment. Built with modularity in mind, it allows for deterministic, automated security workflows.

## 🚀 Features
- **Deterministic Orchestration**: Rule-based logic engine for reliable scan workflows.
- **Modular Agents**: Decoupled modules for recon, auditing, and exploitation.
- **Automated Data Pipeline**: Structured JSON output for easy analysis and reporting.
- **Streamlit Dashboard**: Centralized Command & Control (C2) interface.
